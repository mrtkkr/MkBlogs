Gizlilik Politikası

1. Toplanan Bilgiler

  • Kişisel Bilgiler: Ad, e-posta adresi, telefon numarası vb.
  • Otomatik Bilgiler: IP adresi, çerezler, tarayıcı bilgileri.

2. Bilgilerin Kullanımı

Toplanan veriler, hizmetlerimizi geliştirmek ve yasal yükümlülükleri yerine getirmek amacıyla kullanılabilir.

3. Çerezler

Web sitemiz, kullanıcı deneyimini artırmak için çerezler kullanabilir. Çerezler, ziyaret ettiğiniz web sayfalarını, giriş bilgilerinizi, dil tercihleriniz gibi bilgileri saklamak için tarayıcınızda depolanabilir.

  • İşlevsel Çerezler: Web sitesinin temel işlevlerini yerine getirmesini sağlar.
  • Analitik Çerezler: Web sitesi performansını izlemek ve geliştirmek amacıyla kullanılır.
  • Reklam Çerezleri: Kullanıcıların ilgi alanlarına uygun reklamlar göstermek için kullanılır.

Çerezleri kabul etmek veya reddetmek için tarayıcı ayarlarınızı kullanabilirsiniz. Çerezleri devre dışı bırakmak, web sitesinin bazı özelliklerinin düzgün çalışmamasına neden olabilir.

4. İletişim

Gizlilik politikamız hakkında sorularınız için bizimle iletişime geçebilirsiniz.
